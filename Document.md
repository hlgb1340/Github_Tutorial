# Blueprint basic

### Step 1: Init project

- git init

### Step 2: Create branch

- git checkout -b <branch name>

### Step 3: Commit code

- git add . -> add all file | git add <name file> -> add name file
- git commit -m "message" -> commit code

### Step 4: Add remote

- git remote add origin <link repo ssh>

### Step 5: Push code

- git push origin <branch name>
